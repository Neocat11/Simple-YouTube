# Contributing

Thanks for helping improve Simple-YouTube.

## Contribution Rules

- Keep the extension single-purpose: reduce YouTube visual distraction.
- Keep permissions minimal.
- Do not add analytics, telemetry, tracking, or external network requests.
- Add or update tests when selector behavior changes.
- Add or update fixtures when supporting a new YouTube surface.
- Include before/after notes for affected YouTube pages in pull requests.

## Development

```powershell
npm install
npm test
```

# Privacy Policy

Last updated: 2026-04-30

Simple-YouTube is designed to work locally in your browser.

## Data Collection

Simple-YouTube does not collect, transmit, sell, or share personal data.

The extension does not collect:

- browsing history
- YouTube watch history
- search queries
- account information
- analytics or telemetry
- page contents for external processing

## Local Storage

Simple-YouTube stores one local setting:

- whether Simple-YouTube is enabled or disabled

This setting is stored with `chrome.storage.local` and is used only to apply or disable the extension's visual behavior.

## Network Requests

Simple-YouTube does not make external network requests.

## Third Parties

Simple-YouTube does not use third-party analytics, advertising, tracking, or remote configuration services.

## Changes

If the extension's data behavior changes, this document and the Chrome Web Store privacy disclosures must be updated before release.
